# Lista de Exercícios em C - Estruturas Condicionais

Repositório com resoluções de exercícios da disciplina de Algoritmos, implementados em linguagem C.

## Objetivo

Organizar e publicar minha evolução nos estudos de programação em C, praticando:

- estruturas condicionais com `if`, `else if` e `else`
- estrutura de seleção `switch`
- entrada e saída de dados
- operadores relacionais e lógicos
- lógica de programação

- Exercício 02 - Verificação de voto

- Exercício 04 - Cálculo do valor de maçãs

- Exercício 06 - Cálculo de peso ideal

- Exercício 09 - Maior entre três valores
